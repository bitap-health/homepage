# BitaP Webサイト 制作計画書

## 概要

Google Sites（https://sites.google.com/view/bitap-health）の構造・文言を元に、
GitHub + Netlify で自動デプロイされる静的サイトを作成する。

---

## ファイル構成

```
web/
├── plan.md          ← このファイル
├── index.html       ← トップページ（全セクション含む）
├── style.css        ← スタイル
└── images/          ← 画像（当面はダミー）
```

## デザイン方針

- Google Sitesのシンプルな白ベースを模倣
- フォント：sans-serif系（日本語はシステムフォント）
- カラー：白背景 + 黒テキスト + アクセントカラー（緑系を想定）
- レイアウト：シングルカラム中心、セクションごとに区切り
