# お知らせ追加・編集マニュアル

## ファイル構成

```
news/
└── 20260601/
    ├── content.md   ← 人が読み書きするファイル（内容確認・指示出しに使う）
    ├── index.html   ← Claudeが生成するファイル（直接触らない）
    └── images/      ← この記事で使う画像
```

---

## 新規記事の追加手順

### STEP 1：箱を作る（Claudeに依頼）

日付とタイトルだけ伝えればOKです。

```
お知らせの箱を作って。

日付：2026/06/01
タイトル：【事業事例】〇〇を実施しました
```

Claudeが行うこと：
- `news/20260601/` フォルダ・`images/` を作成
- 空の `content.md` を作成
- `index.html` を生成
- `news.json` に追記
- GitHubにpush

---

### STEP 2：本文を書く（content.mdを直接編集）

作成された `news/20260601/content.md` をテキストエディタで開いて、本文を書き込みます。

**content.mdの書き方：**

```
---
date: 2026/06/01
title: 【事業事例】〇〇を実施しました
thumbnail: 20260601_1.jpg
---

本文テキスト...

![](images/20260601_2.jpg)

続きのテキスト...
```

画像ファイルは `news/20260601/images/` に保存してください。

画像の書き方は `_template/news-template.md` を参照してください。

---

### STEP 3：サイトに反映（Claudeに依頼）

```
20260601のcontent.mdを見てindex.htmlを更新してpushして
```

これでNetlifyに自動反映されます。

---

## 既存記事を編集する

`content.md` を直接書き換えてから、Claudeに反映を依頼します。

```
20260601のcontent.mdを見てindex.htmlを更新してpushして
```

または口頭で伝えることもできます：

```
20260601の記事の〇〇の部分を〇〇に変えてください。
```

---

## よくある操作

| やりたいこと | 手順 |
|-------------|------|
| 記事を新規追加 | STEP 1→2→3 |
| 本文を修正 | content.mdを編集 → STEP 3 |
| 画像を追加 | images/に保存 → content.mdに追記 → STEP 3 |
| 記事を削除 | 「20260601の記事を削除して」とClaudeに伝える |
| サムネイルを変更 | content.mdのthumbnail行を書き換え → STEP 3 |
| 内容を確認 | content.mdを開く（HTMLより読みやすい） |
